## Fitur Website

- **Desain Modern**: Tema biru dongker dengan panel glassmorphism
- **Responsif**: Tampilan menyesuaikan semua ukuran layar
- **Navigasi Lengkap**: 6 halaman utama (Home, Games, Top Up, Invoice, About, Contact)
- **Form Top Up**: Pilih game, masukkan ID, pilih nominal dan metode pembayaran
- **Invoice Digital**: Tampilan invoice lengkap dengan status pembayaran
- **Search Game**: Fitur pencarian game di halaman Games
- **Animasi Halus**: Efek scroll dan hover yang menarik

## Cara Menggunakan

1. Download semua file ke dalam folder `Y15x-topup-website/`
2. Pastikan struktur folder sesuai dengan di atas
3. Tambahkan gambar-gambar yang diperlukan di folder `assets/images/` dan `assets/images/games/`
4. Buka `index.html` menggunakan browser

## Catatan

- File gambar menggunakan placeholder (nama file harus sesuai dengan yang disebutkan)
- Untuk production, ganti semua gambar placeholder dengan gambar asli
- Backend belum terintegrasi (masih frontend saja)
- Fitur download PDF pada invoice masih simulasi

## Teknologi
